```scss
html {
  font-family: var(--font-family);
  font-size: 10px;
}

body {
  background-color: var(--color-surface-primary);
  letter-spacing: 0.25px;
  color: var(--color-text);
  line-height: var(--lh-body);
}
```

Keep the rest of the file (reset rules for *, ul, ol, a, fieldset, input, textarea, button) unchanged — those are structural resets, not themed. EXCEPT: the `input, textarea ::placeholder` font-family should become `var(--font-family)`.

**In `_chat.scss`:**
Replace hardcoded SCSS color variables with CSS custom properties throughout the file. Specific replacements:

6. `.message--user .message__content` — `color: $white-2` becomes `color: var(--color-bubble-user-text)`, `background-color: $dark-purple` becomes `background-color: var(--color-bubble-user)`

8. `.message--user .editing__textarea` — `@include text($white-2, ...)` becomes `@include text(var(--color-text), ...)`, placeholder `$gray-300` becomes `var(--color-text-muted)`

Keep all structural properties (display, flex, position, padding, margin, width, height, z-index, overflow, transform, breakpoints) exactly as they are. Only change color-related properties.
